Gambar 3.1.a Source Code Asli
Kode dimulai dengan !DOCTYPE html> untuk mendefinisikan jenis dokumen sebagai HTML5, dan html lang="en"> menunjukkan bahwa bahasa dokumen ini adalah bahasa Inggris. Di dalam head>, meta charset="UTF-8"> menetapkan set karakter dokumen ke UTF-8 untuk mendukung berbagai karakter, sementara meta name="viewport" content="width=device-width, initial-scale=1.0"> memastikan halaman web responsif di berbagai perangkat. Tag title>Document/title> memberikan judul "Document" pada halaman yang muncul di tab browser, dan link rel="stylesheet" href="style.css"> menghubungkan dokumen dengan file CSS eksternal bernama "style.css" untuk menata tampilan halaman. Bagian body> memuat konten utama halaman. Di dalamnya, terdapat div id="calculator" sebagai wadah utama kalkulator, dengan input field id="display" yang menampilkan angka dan hasil kalkulasi, serta bersifat readonly agar pengguna tidak bisa mengetik langsung di dalamnya. Div id="keys" berisi tombol-tombol kalkulator. Setiap tombol memiliki atribut onclick yang memanggil fungsi JavaScript tertentu untuk menambahkan karakter ke display, seperti appendToDisplay('+') untuk tombol tambah. Tombol-tombol angka dan operator lainnya (7, 8, 9, -, 4, 5, 6, *, 1, 2, 3, /, 0, .) berfungsi serupa. Tombol = memanggil fungsi calculate() untuk menghitung ekspresi di display, dan tombol C memanggil fungsi clearDisplay() untuk membersihkan display.

3.3 Source Code JavaScript

3.3.a Source Code Asli

const display = document.getElementById("display");

function appendToDisplay(input){
    display.value += input;
}

function clearDisplay(){
    display.value = "";
}

function calculate(){
    try{
        display.value = eval(display.value);
    }
    catch(error){
        display.value = "Error";
    }
}
Gambar 3.3.a Source Code Asli
Source code ini merupakan kode java script asli dari sumber refrensi yang kami ambil, dimana kode ini berguna untuk menghubungkan logika java script dengan elemen HTML pada laman web, dengan menggunakan variable display dan fungsi-fungsinya seperti, appendToDisplay(input), clearDisplay(), dan juga calculate() kita dapat mengontrol dan memanipulasi tampilan dan data di laman web secara dinamis. variable display mengambil elemen HTML dengan id “display”, ini berupa area input teks dimana user memanipulasi data. Fungsi appendToDisplay(input) untuk menerima input dan akan ditambahkan ke dalam elemen dengan ID “display”. F ungsi clearDisplay(), untuk menghapus semua isi dari elemen dengan ID “display”, jadi ketika dipanggil berguna untuk mengosongkan tampilan teks. Fungsi calculate () untuk mengalkulasi ekspresi matematika yang ada dalam elemen dengan ID “display”. Jika berhasil, hasilkan akan ditampilkan di elemen “display”.

3.3.b Source Code Modifikasi

const display = document.getElementById("display");

function appendToDisplay(input) {
    display.value += input;
}

function clearDisplay(){
    display.value = "";
}

function backspace() {
    display.value = display.value.slice(0, -1);
}

function calculate() {
    try {
        display.value = eval(display.value);
    } catch (error) {
        display.textContent = "Error";
    }
}
function calculateSquareRoot() {
    try {
        display.value = Math.sqrt(eval(display.value));
    } catch (error) {
        display.textContent = "Error";
    }
}
Gambar 3.3.b Source Code Modifikasi
Modifikasi source code JS pada program kami yaitu terdapat penambahan fungsi backspace() pada baris ke-11 yang digunakan untuk menghapus nilai pada kalkulator. Perintah slice(0, -1) pada baris ke-12 adalah metode string JavaScript yang mengembalikan bagian dari string. Slice(0, -1) berarti mengambil substring dari indeks 0 hingga satu karakter sebelum akhir string. Dengan kata lain, ini menghilangkan karakter terakhir dari string. Hasil dari slice(0, -1) kemudian ditetapkan kembali ke display.value, sehingga teks di tampilan diperbarui dengan satu karakter terakhir dihapus. Lalu terdapat penambahan fungsi yaitu pada baris ke-22 yang dimana merupakan fungsi untuk menghitung akar kuadrat dari nilai yang ada di display. Pertama, display.value dievaluasi menggunakan eval() untuk mendapatkan nilai matematika dari string. Kemudian, fungsi Math.sqrt() digunakan untuk menghitung akar kuadrat dari nilai tersebut. Hasilnya kemudian diperbarui ke display.value.
