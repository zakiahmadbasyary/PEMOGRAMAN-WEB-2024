Materi HTML Tag Paragraf

Source Code

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Example</title>
</head>
<body>
    <p>Ini adalah homepage pertama
        saya, karena saya baru  belajar
        tentang cara membuat homepage.</p>
    <p>Homepage ini masih dalam
        tahap pengembangan, oleh karena itu
        tampilannya masih terlalu sederhana.</p>
    <p>Saya akan berusaha untuk terus
        memperbaiki homepage saya ini,
        sehingga semakin lama semakin
        menarik untuk dilihat.</p>
</body>
</html>

Output

Ini adalah homepage pertama saya, karena saya baru belajar tentang cara membuat homepage.

Homepage ini masih dalam tahap pengembangan, oleh karena itu tampilannya masih terlalu sederhana.

Saya akan berusaha untuk terus memperbaiki homepage saya ini, sehingga semakin lama semakin menarik untuk dilihat.

penjelasan

Teks HTML tersebut berisi tiga paragraf yang menjelaskan bahwa ini adalah homepage pertama yang dibuat oleh penulis, yang masih dalam tahap pembelajaran dan pengembangan. Paragraf pertama menyatakan bahwa ini adalah upaya awal penulis dalam membuat homepage. Paragraf kedua mengakui bahwa tampilan saat ini masih sangat sederhana karena masih dalam pengembangan. Paragraf ketiga menunjukkan komitmen penulis untuk terus memperbaiki dan memperindah homepage tersebut agar semakin menarik seiring berjalannya waktu.
