Materi JavaScript

Source Code

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Example</title>
</head>
<body>
    <button onclick="changeText()">Klik untuk mengubah teks</button>
    <script>
        function changeText() {
            document.getElementById("text").innerHTML = "Teks telah diubah!";
        }
    </script>
</body>
</html>

Output

Ini adalah teks awal.

penjelasan

Kode HTML dan JavaScript ini menampilkan paragraf dengan teks awal "Ini adalah teks awal." dan sebuah tombol yang, ketika diklik, akan memicu fungsi JavaScript changeText(). Fungsi ini mengubah isi elemen paragraf dengan id "text" menjadi "Teks telah diubah!" menggunakan metode innerHTML. Dengan demikian, kode ini mendemonstrasikan cara mengubah konten teks dalam elemen HTML secara dinamis melalui JavaScript berdasarkan interaksi pengguna, yaitu klik tombol.
